eyesight
========

*Version 1.0.0*

Simple script to enable/disable the built-in camera in OS X.


Requirements
------------

* OS X 10.9.x+
* [Homebrew](https://brew.sh)


Installation
------------

```
brew tap lojoja/main
brew install eyesight
```


Use
---
Enable the camera:

```
eyesight enable
```

Disable the camera:

```
eyesight disable
```
